What is HTML?

HTML stands for Hyper Text Markup Language. HTML documents are the heart of the Web.

HTML is based on:
- Text
- Markup
- References to other documents (e.g. pictures and videos)
- Hyperlinks

What is Markup?

Markup is used to render plain text. Specific tag elements can be used to display different text styles and text formatting.

Tags

Most tags contain an opening tag and a closing tag. Here are two common tags:
The < b > tag is used to change the fond to bold and is closed with < /b >
The < em > tag is used to change the fond to italic and is closed with < /em >

Attributes

A tag can have muliple attributes. An example of a tag using an attribute is the anchor tag. The anchor tag is used to create links to other websites.
This link explains how the achor tag works.

How to add images

Images are added using the following tag:
< img src="URL" alt="text" >
SRC stands for source and contains the link to where the picture is stored.
ALT stands for alternative and the text in the quotes is displayed if the piture cannot be found and is read out to blind people.

Inline and block tags

Inline tags are plain text elements. For example the < br > tag can be used inbetween text to start a new line.
Block tags create a box around the element which can have hight and width. For example the < p > tag creates a new paragraph box which needs to be closed with a closing tag ( < /p >).
